git Basic Commands

  • To add/track single file use : git add [fileName here]
  • To add/track all files together use : git add .
  • To commit/store files in local repo use : git commit -m "[Here your sepcific message]"
  • To push these changes to github or server repo use : git push origin [branch]

Git Branches

  • To create branch : git branch [branch name]
  • Switch to other branch : git checkout [branch name]
  • To check list of branches: git branch
  • To merge changes to main branch: go to branch where you want to merge changes (e.g main) : git branch merge [branch name]
  • To delete the branch after completion or whenever [this will delete locally]: git branch -d [branch name]
  • To delete the branch after completion or whenever [to delete from online]: git push origin --delete [branch name]
    • NOTE: After delete the branch make sure to push changes to online repo as well by git push origin main
    NOTE: After delete the branch make sure to push changes to online repo as well by git push origin main

Git Pull and Merge commands

  • To get the or merge the main branch changes to child branch Use: Git pull origin [branch name]
  • To get the or merge the main branch changes to child branch Use: Git pull origin main

Git Rebase cmd

Git rebase and merge command almost similar Mainly git rebase is used to make the child branch similar to main branch to avoid conflict

We can say that with rebase child branch redirected to main branch
  • Command to use git rebase : Firstly rebase the main branch in child branchgit rebase main
  • 2nd step also rebase the main branch with child branch : git rebase [child branch]

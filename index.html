git Basic Commands

  • To add/track single file use : git add [fileName here]
  • To add/track all files together use : git add .
  • To commit/store files in local repo use : git commit -m "[Here your sepcific message]"
  • To push these changes to github or server repo use : git push origin [branch]

Git Diff


If you want to see the diff between current code or last commited or tracked code then you can use
git diff : This will show you difference between current code and last code


If you want to see the difference between particular code or commit then you below command
git diff [id of first commiit/tracked code ] [id of second]

git restore [filename] , for the normal cases like if file is in unstracked or not in staging area git restore --staged [filename] : This will move file from staging to working directory NOTE : This command also usedfull to move file back to working direc from staging area
